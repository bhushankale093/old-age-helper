# **OldAge Helper**

OldAge Helper is an app for helping people above age 65. This is the prototype for the app.


## Features

---

- Medicine Reminder
- Helpline Numbers
- Important Dates
- Account Details
- We can add entries by clicking on plus button which is at bottom right corner
